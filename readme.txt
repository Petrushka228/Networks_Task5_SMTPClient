Задача №5 - SMTP клиент.

По пути Networks_Task5_SMTPClient\SMTPClient\bin\Debug\netcoreapp3.1 лежит exe-файл, который запускает сервер.
В этой же директории лежит файл Config.txt, Body.txt(текст сообщения) и файлы вложений.
Config.txt имеет следующую струкуру: первая строка - тема сообщения, вторая строка - адреса получаелей через пробел, третья - имена файлов вложений через пробел.

При запуске сервера вывоится вся необходимая информация. 

P.S. Может возникать ошибка подключения к тправителю - из-за блокировки самого ящика от посторонних приложений, можно отключить в настройках своего ящика.
